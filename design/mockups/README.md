# design/mockups

Raw HTML exported from Claude Design lands here, one file per screen
(e.g. `backlog-board.html`, `approvals-inbox.html`, `finance-dashboard.html`).

Rules:
- These files are **reference only** — never imported or shipped by the app.
- Never hand-edit them; if a screen changes, re-export and overwrite.
- Keep them committed. They're the audit trail for "why does this token exist,"
  and the input to `scripts/extract-tokens.js`.
